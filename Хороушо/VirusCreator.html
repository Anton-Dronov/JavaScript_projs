JavaScript Tests

Šis JavaScript koda piemērs demonstrē, kā apgriezt ievadīto teksta virkni.

Tas tiek veikts vienā garā komandā, kas sastāv no trim soļiem(1):

  1. Sadaliet visu izteiksmi atsevišķos simbolos: mansTeksts.split("");
  2. Apgrieziet simbolu secību: mansTeksts.reverse();
  3. Pievienojiet simbolus vienā izteiksmē: mansTeksts.join("");

Kopumā tas izskatās šādi: virkne.split("").reverse().join("");

Ievadiet savus skaitļus augšējos laukos un noklikšķiniet uz pogas, lai izpildītu JavaScript koda funkciju. Rezultāts parādīsies apakšējā laukā.
