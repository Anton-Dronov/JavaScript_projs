Problēmas apspriešana

Parasti JavaScript algoritms šai problēmai izskatās ļoti vienkārši:

  • Pārbaudiet ievadītās vērtības,
  • Aprēķināt LKD vērtību,
  • Izdrukāt rezultātu.

Skatiet blokshēmu, kas sniegts kreisajā sadaļā.

Vissarežģītākā šī koda daļa ir pirmais bloks. Tas sastāv no divām daļām:

  • Pārbaudiet, vai kaut kas ir ievadīts, un ja tas bija, vai tā bija skaitļa vērtība, vai nē.
  • Pārbaudiet, vai šī numura bāze ir izvēlēta pareizi.

Lai veiktu šo pārbaudi, jums ir jāsadala skaitlis, kas ievadīts, atsevišķos ciparos, un rezultāts jāsaglabā jaunajā masīvā. Pēc tam pārbaudiet katru šī masīva elementu (ciparu) attiecībā pret izvēlētās bāzes vērtību. Ja visas šīs pārbaudes izturēsies, sāksies konvertēšanas process(2).
